# ReparaYa

### Plataforma que conecta clientes que necesitan un servicio con contratistas que ofrecen dicho servicio (p. ej., mantenimiento, reparaciones).

Marketplace web con: búsqueda/descubrimiento, perfiles, agenda y reservas, pagos con anticipo, check-in y liquidación al completar, calificaciones, mensajería básica, y un panel administrativo mínimo.
